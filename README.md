# AI Emoji Generator 🎨

A full-stack web application that generates custom emojis from user prompts using AI.

## Features
- AI-generated emoji from text
- Free pre-trained image generation model
- Python Flask backend
- HTML, CSS, JavaScript frontend
- No datasets, no paid APIs

## Tech Stack
- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- AI: Pollinations.ai (free image generation API)
